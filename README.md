# cc-timestamps

A Claude Code plugin marketplace containing the **cc-timestamps** plugin.

## Installation

```bash
# 1. Add this marketplace
claude plugin marketplace add https://github.com/seannam/cc-timestamps

# 2. Install the plugin
claude plugin install cc-timestamps@cc-timestamps
```

## Plugin

### cc-timestamps

Displays timestamps on every message with retro IM styling, duration tracking, and session statistics.

See [plugins/cc-timestamps/README.md](plugins/cc-timestamps/README.md) for full documentation.
